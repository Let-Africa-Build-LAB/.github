# Let Africa Build (LAB)

**Africa's First Bitcoin Focused Developer Ecosystem & Builder Incubator**

LAB is an open source developer program incubating Africa's next generation of Bitcoin builders. We provide structured curriculum, funding pathways, and community infrastructure to grow the continent's Bitcoin developer talent pipeline.

### Key Repositories

| Repository | Description |
|---|---|
| [about-LAB](https://github.com/Let-Africa-Build-LAB/about-LAB) | Institutional overview, proof of work, and program model |
| [lab-bitcoin-curriculum](https://github.com/Let-Africa-Build-LAB/lab-bitcoin-curriculum) | 6-month Bitcoin developer bootcamp curriculum |
| [lab-open-source-builders-fund](https://github.com/Let-Africa-Build-LAB/lab-open-source-builders-fund) | Open source builders fund backed by Artizen ($20K matching pool) |
